# API

API using ExpressJS.

## Dependencies

* ExpressJS
* SequelizeJS

## Installation / Setup

* Install dependencies `npm install`
* Run server `DEBUG=api:* npm start`
* Visit `http://localhost:3000/`

### Testing

* Run `npm test` (note: server from above must be running)

## End points

### Tags

* [GET] `/tags`
* [POST] `/tags`

### Posts

* [GET] `/posts`
* [POST] `/posts`
